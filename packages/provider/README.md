# @openmerch/provider

> **Internal / preview — not part of the current public SDK surface.**
>
> Provider integrations on OpenMerch are currently operator-managed. There is no public provider onboarding or self-serve provider registration today, and this package is not intended for external use. It remains in the repo as a placeholder for the provider type contracts while the public surface is narrowed to the agent SDK.

## Scope

This package contains TypeScript type definitions used internally to model the provider side of the OpenMerch network — service definitions, pricing, and execution handler shapes. It exposes no runtime client and no registration helpers.

If you are an agent author, use [`@openmerch/agent`](../agent) instead.

## Pre-1.0 Stability

This package is versioned below 1.0 and may change or be removed without notice while the public surface stabilizes.
